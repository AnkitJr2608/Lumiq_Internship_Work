Response Body
For creating a table:- A table for userprofile gets created with response output as "Table created successfully"

For adding a user in the table:- It asks for all the details along with their validation checks and if everything gets validated successfully then it enters the user values in the table along with the success message
"User successfully created."

For updating a user in the table:- It asks for the user ID and the attribute and its value to be changed. If the ID is present then the attribute gets changed and the success message displays as "User successfully updated". If the ID entered is 
   not present then a error message displays as "User not found"
